# Assets Directory

Place your logo and other static assets here.

## Logo File
- **File name**: `logo.png` (or `logo.svg`, `logo.jpg`)
- **Recommended size**: 512x512px for best quality
- **Location**: `/public/assets/logo.png`

The logo will be used for:
- Browser favicon
- Header logo
- Footer logo
- App icon (mobile)

## Supported Formats
- PNG (recommended for logos with transparency)
- SVG (scalable vector graphics)
- JPG/JPEG (for photos)

## Usage
Once you add your logo file to this directory, it will be automatically available at `/assets/logo.png` in your application.
